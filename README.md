
##这个项目是什么?
一个在线o2o类型的团购网站

界面风格仿照糯米网


##o2o_ByThinkPHP5.0.3有哪些功能？

* 强大的`商家`功能，方便定制自己的产品
    *  商家后台地址：http://localhost/public/bis/index/index.html,当然首页也有入口
    *  商家入住申请后，需管理员审核
    *  可以发布商品分类信息
    *  可以管理商家、商家发布的产品、商家的分店等
* 管理员模块
    *  管理员后台地址：http://localhost/public/admin/index/index.html
    *  可以发布商品分类信息
    *  可以审核商家、商家发布的产品、商家的分店等
* 前台展示模块
* 购买功能，支持微信扫一扫
* 更多功能后续开发
